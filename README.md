- Purchase listings through the shopping cart.
  
Sellers:
- Create new car listing with car image, price, and details.
- Remove and update listings.

Admins:
- Add credits to users accounts through Admin dashboard.
- Change users account type to either User, Seller, or Admin through Admin dashboard.
- Delete/edit other user's listings incase of abuse.
